## [发现带有假 AGP 插槽的仿冒复古主板](https://hackaday.com/2026/07/21/counterfeit-retro-mainboards-with-fake-agp-slots-are-a-thing/) ⭐️ 7.0/10

复古计算爱好者[Computer Retro Bus]发现了一些仿冒的复古主板，这些主板带有假的 AGP 插槽，实际上并不工作。该发现由 Hackaday 报道，这些主板旨在欺骗那些组装 Pentium 4 时代系统的收藏者。 这凸显了复古硬件市场中仿冒品日益严重的问题，可能浪费收藏者的金钱并损害信任。对于追求原汁原味组装的复古爱好者来说，硬件的真实性至关重要。 假的 AGP 插槽物理上存在，但缺乏电气连接，纯粹是装饰性的。这些仿冒主板针对 Socket 478 主板，该插槽用于 2001 年至 2004 年的 Intel Pentium 4 和 Celeron CPU。

rss · Hackaday · 7月21日 20:00

**背景**: AGP（加速图形端口）是 1990 年代末至 2000 年代中期用于 PC 显卡的专用插槽，后来被 PCI Express 取代。Socket 478 是 Intel 用于 Pentium 4 处理器的 CPU 插槽。随着对复古 PC 零件需求的增加，仿冒复古主板变得越来越常见。

## [近 30 年后，Java 戒指被成功修复](https://hackaday.com/2026/07/21/java-ring-restored-after-nearly-30-years/) ⭐️ 7.0/10

**干啥的**: Java 戒指是 1990 年代末 Sun Microsystems 推出的一款可穿戴 Java 设备，基于达拉斯半导体的 iButton 技术，内置微处理器和 Java 虚拟机，可用于身份认证和运行小型 Java 程序。

**亮点**: - 成功修复一枚近 30 年前的稀有 Java 戒指，展示了复古硬件修复的技术可能性。
- 涉及 1-Wire 协议逆向工程和固件恢复，对硬件黑客有技术参考价值。
- 作为早期可穿戴 Java 设备，具有重要的历史和技术怀旧意义。

**我的判断**: 值得关注。对于有硬件维修经验的工程师来说，这个修复案例展示了如何复活老式专有设备，尤其是 1-Wire 通信和 Java 虚拟机的恢复过程。虽然实际复刻意义不大，但学习其修复思路和逆向方法对理解早期嵌入式 Java 系统有帮助。

rss · Hackaday · 7月21日 11:00

**背景**: Java 戒指是 1990 年代末 Sun Microsystems 在 Java 开发者大会上分发的一种可穿戴计算机。它采用了达拉斯半导体的 iButton 技术，这是一种 16 毫米不锈钢钥匙扣，通过 1-Wire 串行总线进行低速数据通信和供电。戒指可以存储和运行小型 Java 小程序，是可穿戴 Java 设备的早期范例。

- **仿冒主板案例**：假 AGP 插槽的电气检测思路可直接迁移到电源维修中——例如通过测量假负载、假散热片或假电容的电气连接，快速判断元件是否真实工作，避免误判故障点。
- **Java 戒指修复**：1-Wire 协议逆向与固件恢复过程，能帮你理解“低功耗通信+嵌入式系统”的电源管理逻辑，对学习 PFC/LLC 控制芯片的供电时序和待机回路设计有参考价值。
- **数据中心冲突报道**：AI 数据中心对电力和冷却的巨大需求，间接强调了高效电源（如 PFC 和 LLC 拓扑）在降低能耗、提升功率密度中的关键作用，可作为你学习电源设计时理解“效率为何是核心指标”的现实案例。
- **树脂 3D 打印回顾**：SLA 打印机中 UV LED 光源的驱动电路与开关电源的恒流驱动原理相通，维修此类设备时，你已有的电源测试经验（如纹波、负载调整率）可直接用于诊断打印质量异常。
